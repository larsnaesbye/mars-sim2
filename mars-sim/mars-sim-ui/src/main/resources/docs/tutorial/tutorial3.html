Tutorial - Step 3: Menu and toolbars

At the top of the Main Simulation Window, just below the window header, is the menu bar, with the menu headings: 'File', 'Tools', and 'Help'.

File deals with the functions you are used to in most menu-driven programs: Exit the program, make a new document (here: a simulation), and open and save.
